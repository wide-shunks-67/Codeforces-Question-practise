<div class="header"><div class="title">B. Simply Sitting on Chairs</div><div class="time-limit"><div class="property-title">time limit per test</div>1.5 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>  </p><p>There are $$$n$$$ chairs in a row, initially all unmarked.</p><p>You are given a permutation $$$p$$$$$$^{\text{∗}}$$$ of length $$$n$$$.</p><p>Now, you play a game. You visit each chair sequentially, starting from the $$$1$$$-st chair. At the $$$i$$$-th chair, you can do the following: </p><ul> <li> If the $$$i$$$-th chair is already marked, then you end the game immediately without sitting on it. </li><li> Otherwise, you can choose to sit on the chair or skip it and move to the next chair. </li><li> If you choose to sit on the chair, then after standing up, you mark the $$$p_i$$$-th chair and move to the next chair. </li></ul> If all the $$$n$$$ chairs are visited, the game ends.<p>Your task is to determine the maximum number of chairs that you can sit on.</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$A permutation of length $$$n$$$ is an array consisting of $$$n$$$ distinct integers from $$$1$$$ to $$$n$$$ in arbitrary order. For example, $$$[2,3,1,5,4]$$$ is a permutation, but $$$[1,2,2]$$$ is not a permutation ($$$2$$$ appears twice in the array), and $$$[1,3,4]$$$ is also not a permutation ($$$n=3$$$ but there is $$$4$$$ in the array). </p></div></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows. </p><p>The first line of each test case contains a single integer $$$n$$$ ($$$1 \le n \le 2\cdot 10^5$$$) — the number of chairs.</p><p>The second line of each test case contains $$$n$$$ distinct integers $$$p_1, p_2,\ldots,p_{n}$$$ — the permutation $$$p$$$.</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$2\cdot10^5$$$. </p></div><div class="output-specification"><div class="section-title">Output</div><p>Output a single integer — the maximum number of chairs you can sit on.</p></div><div
