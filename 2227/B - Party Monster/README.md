You are allowed to perform the following operation <span class="tex-font-style-bf">at most once</span>: </p><ul> <li> Choose a substring$$$^{\text{∗}}$$$ of $$$s$$$ and remove it. Then, you may reinsert the removed characters back into the remaining string one by one. Each character can be placed at any arbitrary position, independently of the others. </li></ul><p>Yousef wants you to determine whether it is possible to obtain a regular bracket sequence$$$^{\text{†}}$$$ after performing the operation at most once.</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$A <span class="tex-font-style-it">substring</span> is a contiguous subsegment of a string. For example, <span class="tex-font-style-tt">"acab"</span> is a substring of <span class="tex-font-style-tt">"abacaba"</span> (it starts in position $$$3$$$ and ends in position $$$6$$$), but <span class="tex-font-style-tt">"aa"</span> or <span class="tex-font-style-tt">"d"</span> aren't substrings of this string. So the substring of the string $$$s$$$ from position $$$l$$$ to position $$$r$$$ is $$$s[l, r]= s_l s_{l+1} \dots s_r$$$.</p><p>$$$^{\text{†}}$$$A <span class="tex-font-style-it">regular bracket sequence</span> is a bracket sequence that can be transformed into a correct arithmetic expression by inserting the characters $$$1$$$ and $$$+$$$ between the original characters of the sequence. For example:</p><ul> <li> bracket sequences $$$\texttt{()()}$$$ and $$$\texttt{(())}$$$ are regular (the resulting expressions are: $$$\texttt{(1)+(1)}$$$ and $$$\texttt{((1+1)+1)}$$$); </li><li> bracket sequences $$$\texttt{)(}$$$, $$$\texttt{(}$$$ and $$$\texttt{)}$$$ are not. </li></ul></div></div><div class="input-specification"><div class="section-title">Input</div><p>The first line contains an integer $$$t$$$ ($$$1 \le t \le 10^4$$$) — the number of test cases.
