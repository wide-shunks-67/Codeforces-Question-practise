<div class="header"><div class="title">B. Beautiful Numbers</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>512 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>Let's define $$$F(x)$$$ as the sum of the digits of $$$x$$$. An integer $$$x$$$ is considered <span class="tex-font-style-it">beautiful</span> if $$$F(F(x)) = F(x)$$$.</p><p>You are given an integer $$$x$$$. In one move, you can choose any digit in the number and replace it with another. The resulting number cannot have leading zeros.</p><p>Your task is to calculate the minimum number of moves (possibly zero) required to make the given number beautiful.</p></div><div
