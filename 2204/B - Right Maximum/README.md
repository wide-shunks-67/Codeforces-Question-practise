class="property-title">output</div>standard output</div></div><div><p>You are given an array $$$a$$$ consisting of $$$n$$$ integers.</p><p>While the array is not empty, an operation is performed consisting of two steps:</p><ul> <li> first, the maximum element in the array is chosen (if there are multiple maximum elements, the rightmost maximum is chosen); </li><li> then, all elements after the chosen element, <span class="tex-font-style-bf">including it</span>, are removed from the array. </li></ul><p>Your task is to calculate the number operations that will be performed before the array becomes empty.</p></div><div class="input-specification"><div class="section-title">Input</div><p>The first line contains one integer $$$t$$$ ($$$1 \le t \le 10^4$$$) — the number of test cases.</p><p>Each test case consists of two lines:</p><ul> <li> the first line contains one integer $$$n$$$ ($$$2 \le n \le 2 \cdot 10^5$$$); </li><li> the second line contains $$$n$$$ integers $$$a_1, a_2, \dots, a_n$$$ ($$$1 \le a_i \le n$$$). </li></ul><p>Additional constraint on the input: the sum of $$$n$$$ over all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div
